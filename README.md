🧑‍💻 Pablo Muñoz – Website
Descripción del proyecto

Este proyecto consiste en el desarrollo de un sitio web profesional para Pablo Muñoz, barbero, educador y creador de sistemas de formación para profesionales de la barbería.

La web fue diseñada para presentar su filosofía profesional, programas educativos, portfolio de trabajos y formaciones, manteniendo una estética moderna, minimalista y alineada con su identidad visual.

El objetivo principal fue construir una plataforma clara, rápida y visualmente sólida que permita a los usuarios conocer su trabajo y acceder fácilmente a sus programas de formación.

🌐 Sitio web

https://pablomunozoka.com

El deploy fue realizado utilizando Netlify, con dominio personalizado configurado desde Porkbun y certificado SSL automático mediante Let's Encrypt.

📂 Estructura del proyecto

El proyecto está organizado siguiendo una arquitectura basada en componentes:

src
 ├── assets        # Imágenes, videos y recursos multimedia
 ├── components    # Componentes reutilizables (Navbar, Footer, etc)
 ├── context       # Context API (gestión de idioma)
 ├── i18n          # Sistema de traducciones
 ├── pages         # Páginas principales del sitio
 │
 ├── App.jsx       # Configuración de rutas
 ├── main.jsx      # Punto de entrada de la aplicación

Esta estructura permite mantener el código organizado, escalable y fácil de mantener.

__________________________________________________

🌍 Sistema de internacionalización

El sitio implementa un sistema simple de internacionalización (i18n) mediante un archivo de traducciones.

El idioma se gestiona mediante React Context, permitiendo cambiar dinámicamente entre español e inglés desde la interfaz.
